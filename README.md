## What is FlipDeck?

FlipDeck puts a classic flipbook animation workflow into the browser, so you can draw frame by frame on a canvas without installing desktop software. The focus is on tactile editing: sketching, timing, and playback all live in one web-based workspace.

It works well for fast animation tests, rough scene blocking, and iterative drawing sessions where nearby frames need to stay visible for reference. Because it runs in the browser, it is quick to open, simple to revisit, and low-friction to use.

---

## Core Features

- Browser-based canvas workflow for sketching animation frames
- Frame-by-frame editing for flipbook-style motion
- Onion-skin support to compare adjacent line work
- Real-time playback to preview timing as you draw
- Frame management tools for duplicating and nudging frames
- Pencil tools for freehand drawing
- Color controls for adjusting the look of your strokes
- No-install web access for quick session starts

## How to Use It

1. Open FlipDeck in a supported browser.
2. Create or select a frame to begin sketching on the canvas.
3. Use onion skinning to compare your current drawing with nearby frames.
4. Duplicate or nudge frames when adjusting motion and spacing.
5. Play the animation in real time to review timing and flow.
6. Refine strokes and color choices until the sequence matches your intent.

Typical workflow:

- rough out key poses
- fill in in-between frames
- preview playback
- adjust frame order or spacing
- repeat until the animation feels right

## FAQ

**Does FlipDeck require installation?**  
No. It is designed for browser-based use.

**Can I see my animation while I draw?**  
Yes. Real-time playback is part of the workflow.

**How do I line up frames?**  
Use onion-skin view to compare adjacent drawings and refine motion.

**Where are frame controls handled?**  
Frame actions such as duplication and nudging are managed inside the app.

**What if something does not load correctly?**

Use the latest release or build from the project source when you want the current version.
